# 虎博客后台管理系统

## 功能特性

### 🎯 核心功能
- **文章管理**: 创建、编辑、删除博客文章
- **页面管理**: 管理关于、友链、留言等页面
- **媒体管理**: 上传和管理图片、文件
- **友情链接**: 管理博客友链
- **说说动态**: 发布和管理动态内容
- **个人资料**: 更新个人信息
- **网站设置**: 配置网站基本信息
- **数据统计**: 查看访问统计和分析

### 🎨 界面特性
- **现代化设计**: 基于 Material Design 的现代化界面
- **响应式布局**: 支持桌面端和移动端
- **深色/浅色主题**: 支持主题切换
- **实时预览**: 编辑时实时预览效果
- **拖拽上传**: 支持拖拽上传文件

### 🔐 安全特性
- **身份验证**: 基于 Netlify Identity 的身份验证
- **权限控制**: 细粒度的权限管理
- **数据备份**: 自动备份重要数据
- **操作日志**: 记录所有操作日志

## 快速开始

### 1. 访问管理后台
访问 `https://your-domain.com/admin/` 进入管理后台

### 2. 登录系统
- 首次访问需要注册账号
- 使用邮箱和密码登录
- 支持第三方登录（GitHub、Google等）

### 3. 开始使用
登录后即可开始管理博客内容

## 详细使用指南

### 文章管理

#### 创建新文章
1. 点击左侧导航栏的"文章管理"
2. 点击"新建文章"按钮
3. 填写文章信息：
   - **标题**: 文章标题
   - **发布日期**: 发布时间
   - **文章ID**: 唯一标识符
   - **分类**: 文章分类
   - **标签**: 文章标签（可多选）
   - **封面图片**: 文章封面
   - **是否推荐**: 是否在首页推荐
   - **是否置顶**: 是否置顶显示
   - **是否隐藏**: 是否隐藏文章
   - **摘要**: 文章摘要
   - **正文**: 文章内容（支持 Markdown）

#### 编辑文章
1. 在文章列表中找到要编辑的文章
2. 点击"编辑"按钮
3. 修改文章内容
4. 点击"保存"按钮

#### 删除文章
1. 在文章列表中找到要删除的文章
2. 点击"删除"按钮
3. 确认删除操作

### 页面管理

#### 管理页面类型
- **关于页面**: 个人介绍页面
- **友链页面**: 友情链接展示页面
- **留言页面**: 访客留言页面
- **动态页面**: 说说动态展示页面

### 媒体管理

#### 上传文件
1. 点击"媒体管理"
2. 拖拽文件到上传区域或点击选择文件
3. 支持的文件格式：jpg, png, gif, webp, pdf, doc, docx
4. 文件大小限制：10MB

#### 管理文件
- 查看所有上传的文件
- 复制文件链接
- 删除不需要的文件

### 友情链接

#### 添加友链
1. 点击"友情链接"
2. 点击"添加友链"
3. 填写友链信息：
   - **名称**: 友链名称
   - **链接**: 友链地址
   - **头像**: 友链头像
   - **描述**: 友链描述

### 说说动态

#### 发布动态
1. 点击"说说动态"
2. 点击"发布动态"
3. 填写动态内容：
   - **日期**: 发布时间
   - **标签**: 动态标签
   - **内容**: 动态内容（支持 Markdown 和图片）

### 个人资料

#### 更新信息
- **姓名**: 个人姓名
- **头像**: 个人头像
- **简介**: 个人简介
- **邮箱**: 联系邮箱
- **GitHub**: GitHub 链接
- **微博**: 微博链接
- **QQ**: QQ 号码
- **微信**: 微信号

### 网站设置

#### 基本设置
- **网站标题**: 网站标题
- **网站描述**: 网站描述
- **关键词**: SEO 关键词
- **作者**: 网站作者
- **网站图标**: 网站 favicon
- **Logo**: 网站 Logo
- **备案号**: ICP 备案号
- **统计代码**: 第三方统计代码

### 数据统计

#### 查看统计
- **访问量**: 总访问量统计
- **访客数**: 独立访客统计
- **页面浏览量**: 各页面访问统计
- **来源统计**: 访问来源分析
- **设备统计**: 访问设备分析

## 配置说明

### Netlify 配置

#### 1. 启用 Identity
在 Netlify 后台启用 Identity 服务

#### 2. 配置 Git Gateway
启用 Git Gateway 以支持通过 API 修改文件

#### 3. 设置注册
- 选择注册方式（邀请制或开放注册）
- 配置邮件模板
- 设置第三方登录

### 自定义配置

#### 修改主题色
在 `public/admin/index.html` 中修改 CSS 变量：
```css
:root {
  --primary: #01C4B6; /* 主色调 */
  --primary-dark: #00a89b; /* 深色主色调 */
  --primary-light: #e6f7f6; /* 浅色主色调 */
}
```

#### 添加自定义字段
在 `public/admin/config.yml` 中添加新的字段配置

## 常见问题

### Q: 无法登录管理后台？
A: 检查以下几点：
1. 确保 Netlify Identity 已启用
2. 检查邮箱验证是否完成
3. 确认密码是否正确

### Q: 上传文件失败？
A: 可能的原因：
1. 文件大小超过限制
2. 文件格式不支持
3. 网络连接问题

### Q: 文章无法保存？
A: 检查以下几点：
1. 确保 Git Gateway 已启用
2. 检查网络连接
3. 确认文件权限

### Q: 如何备份数据？
A: 所有数据都存储在 Git 仓库中，可以通过以下方式备份：
1. 克隆整个仓库
2. 导出特定文件
3. 使用 Git 历史记录
