📄 文章管理

在这里可以查看、编辑和删除你的博客文章。

✏️ 新建文章

创建新的博客文章，支持 Markdown 格式。

  • 标题：文章的标题
  • 分类：文章的分类，如技术、生活等
  • 标签：文章的标签，用逗号分隔
  • 内容：文章正文，支持 Markdown 语法
  • 封面：文章的封面图片URL（可选）

⚙️ 网站配置

配置网站的基本信息，如标题、描述、作者等。

💡 提示

这是一个简化的管理界面，主要用于内容预览和编辑。实际的文件修改需要通过 Git 进行。
